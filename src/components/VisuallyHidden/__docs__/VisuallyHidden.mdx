# Visually Hidden

A Visually Hidden component is used to hide content from view while maintaining
accessibility for screen readers and assistive technologies. This ensures
inclusivity without impacting the visual design.

## Example

<Canvas of={VisuallyHidden.Default} />

## Usage

```ts
import { VisuallyHidden } from "kaakao";

const SkipLink = () => (
  <VisuallyHidden>
    <a href="#main-content" class="skip-link">Skip to main content</a>
  </VisuallyHidden>
);
```
