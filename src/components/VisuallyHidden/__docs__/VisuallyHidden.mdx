# Visually Hidden

A Visually Hidden component is used to hide content from view while maintaining
accessibility for screen readers and assistive technologies. This ensures
inclusivity without impacting the visual design.

## Example

```
<Canvas of={Button.Primary} />
```

## Usage

```ts
import { VisuallyHidden } from "kaakao";

const SkipLink = () => (
  <VisuallyHidden>
    <a href="#main-content" class="skip-link">Skip to main content</a>
  </VisuallyHidden>
);
```

## Arguments

> Extended from _`HTMLAttributes<HTMLElement>`_

### `as`

_**Type**: `keyof IntrinsicElements` **Default**: `span` **Required**: No_

The HTML element to render the visually hidden content within.

### `children`

_**Type**: `ReactNode` **Default**:`undefined` **Required**: Yes_

The content to be made available to assistive technologies but visually hidden
from the UI.
